Texto de ejemplo: Hojas de Css externas

Con una hoja de estilo externa, puede cambiar el aspecto de un sitio web completo cambiando solo un archivo. Cada página HTML no debe incluir una referencia al archivo de hoja de estilo externo dentro del elemento link en la sección de header

Una hoja de estilo externa se puede escribir en cualquier editor de texto, y debe guardarse con una extensión .css . El archivo .css externo no debe contener ninguna etiqueta HTML.
